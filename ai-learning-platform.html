Project Overview

SoloLeveling AI Career Builder is an innovative web application that uses artificial intelligence to analyze users' resumes and create personalized learning roadmaps for their target tech careers. The platform bridges the skill gap between a user's current qualifications and their desired job role.

Key Features

AI-Powered Career Analysis

Extracts and analyzes text from PDF, DOCX, and TXT files to identify skills and experience

Skills Gap Analysis

Identifies current skills vs. required skills for target careers with detailed insights

Personalized Learning Paths

Generates tailored roadmaps based on individual backgrounds and career goals

Technical Architecture

Frontend Technologies

  • JavaScript (ES6+) for dynamic functionality
  • Chart.js for data visualization

Backend Technologies
